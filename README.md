# Pick Next — Website

Responsive full-stack jewelry website with login/logout support.

Pages included:
- index.html (Home)
- products.html (Products)
- about.html (About Us)
- contact.html (Contact)
- login.html (Login)
- /account (Protected account dashboard)

## Run locally

Install dependencies and start the server:

```bash
cd "c:\Users\User\Desktop\pick next"
npm install
npm start
```

Then open `http://localhost:3000`.
